# Code of Conduct

This project is a learning-focused repository. We expect respectful and professional collaboration.

## Expected Behavior

- Be respectful and considerate in all discussions.
- Assume good intent and be constructive.
- Provide clear and helpful feedback.

## Unacceptable Behavior

- Harassment, hate speech, or discriminatory remarks.
- Trolling, insults, or personal attacks.
- Sharing private information without consent.

## Reporting

If you experience or witness unacceptable behavior, open a GitHub issue describing the concern. The maintainers will review and respond.
